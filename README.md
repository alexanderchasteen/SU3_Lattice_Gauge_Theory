SU3 LGT Code. Included is
- Pure SU3 Gauge Theory simulation. Simulate across couplings for a set thermalization and measure the avg plaq (raw data)
- Analysis of the data called "Tempering_analysis" and "Tempering_graphing" (unrelated to parallel tempering) which returns
  certatain obsevrable measurments
- Polyakov loop calculations to compute the string tension \sigma

The pure SU3 gauge theory simulation has many files required to run it organized by their purpose (statistics, paramters, lattice, rng)
